### このファイルを別のモデルパックに同梱する方へお願い
はじめに、ドメインを変更してください。
ドメインの変更は'assets/minecraft/mc-wrapper'を別の名前に変更すれば良いです。

また、`rmw`, `__rtm_mc_wrapper__`の名前を変更していただきたいです。
名前が競合すると他のかたの作成したスクリプトとの互換性が無くなる恐れがあります。
